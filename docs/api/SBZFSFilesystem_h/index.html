SBZFSFilesystemDefaultMountpoint

SBZFSFilesystemDefaultMountpoint

#define SBZFSFilesystemDefaultMountpoint
Discussion

For clarity of consumer code, this macro may be used to indicate to the initWithNewZFSFilesystem:mountPoint:mounted: method that the default (inherited) mountPoint should be used for the new filesystem that is created.

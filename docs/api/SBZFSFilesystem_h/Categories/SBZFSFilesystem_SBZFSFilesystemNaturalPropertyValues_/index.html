SBZFSFilesystem(SBZFSFilesystemNaturalPropertyValues)

Extends Class: SBZFSFilesystem
Declared In: SBZFSFilesystem.h

Discussion

This category groups convenience methods which present the numerical ZFS filesystem properties in a more natural, user-friendly manner: megabytes and percentages.


Methods

availablePercentage

- (float) availablePercentage;
Discussion

Returns the percentage of the total space alloted to the receiver's ZFS filesystem that is not currently use.

Range is 0.0 to 100.0.

inUsePercentage

- (float) inUsePercentage;
Discussion

Returns the percentage of the total space alloted to the receiver's ZFS filesystem that is actually in use.

Range is 0.0 to 100.0.

quotaMegabytes

- (SBUInteger) quotaMegabytes;
Discussion

Returns the receiver's quota limit in units of megabytes.

reservedMegabytes

- (SBUInteger) reservedMegabytes;
Discussion

Returns the receiver's storage reservation in units of megabytes.

setQuotaMegabytes:

- (BOOL) setQuotaMegabytes:(SBUInteger)megabytes;
Discussion

Calls through to setQuotaByteCount: after converting megabytes to bytes (multiplying by 1024^2).

See setQuotaByteCount: for return values, etc.

setReservedMegabytes:

Discussion

Calls through to setReservedByteCount: after converting megabytes to bytes (multiplying by 1024^2).

See setReservedByteCount: for return values, etc.

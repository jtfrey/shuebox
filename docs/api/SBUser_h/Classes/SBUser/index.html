SBUser

Superclass: SBObject
Declared In: SBUser.h

Discussion

The SBUser class provides a basic (but highly extensible) representation of a user agent. The class is setup to be primarily a wrapper to a mutable dictionary with a pre-defined set of keys for standard user agent properties. Additional convenience methods are present to ease the use of these properties.

authenticateWithPassword:

- (BOOL) authenticateWithPassword:(SBString*)password;
Discussion

Performs simple (cleartext) authentication of the passed-in password against the receiver's SBUserPasswordKey property. If the receiver has no property with this key, then YES is returned (lack of password implies no password necessary, e.g.)

primaryGidForUser

- (gid_t) primaryGidForUser;
Discussion

If the receiver defines the SBUserPrimaryGIDNumberKey property, then its value is converted to a UNIX-native gid_t and returned. If the property is not defined or cannot be converted, -1 is returned.

setUserProperties:

- (void) setUserProperties:(SBDictionary*)properties;
Discussion

Removes all data from the receiver's property dictionary and adds those present in the passed-in properties dictionary.

uidForUser

- (uid_t) uidForUser;
Discussion

If the receiver defines the SBUserUIDNumberKey property, then its value is converted to a UNIX-native uid_t and returned. If the property is not defined or cannot be converted, -1 is returned.

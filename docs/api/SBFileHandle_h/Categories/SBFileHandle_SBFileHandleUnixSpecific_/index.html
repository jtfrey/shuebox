SBFileHandle(SBFileHandleUnixSpecific)

Extends Class: SBFileHandle
Declared In: SBFileHandle.h

Discussion

Methods of SBFileHandle that are particular to Unix OSes.


Methods

-fileDescriptor
-initWithFileDescriptor:
-initWithFileDescriptor:closeOnDealloc:

fileDescriptor

- (int) fileDescriptor;
Discussion

Returns the Unix file descriptor associated with the receiver.

initWithFileDescriptor:

- (id) initWithFileDescriptor:(int)fd;
Discussion

Initialize the receiver to wrap the given file descriptor, fd, with implicit read and write capabilities.

When the receiver is deallocated the file descriptor will NOT be closed.

initWithFileDescriptor:closeOnDealloc:

- (id) initWithFileDescriptor:(int)fd closeOnDealloc:(BOOL)shouldClose;
Discussion

Initialize the receiver to wrap the given file descriptor, fd, with implicit read and write capabilities.

When the receiver is deallocated the file descriptor will only be closed if shouldClose is YES.

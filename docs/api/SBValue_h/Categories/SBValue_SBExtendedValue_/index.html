SBValue(SBExtendedValue)

Extends Class: SBValue
Declared In: SBValue.h

Discussion

Groups additional extensions to the basic SBValue methods.


Methods

-isEqualToValue:
-nonretainedObjectValue
-pointerValue

isEqualToValue:

- (BOOL) isEqualToValue:(SBValue*)value;
Discussion

If the receiver and "value" are of the same type and contain the same byte sequence, return YES. Otherwise, return NO.

nonretainedObjectValue

- (id) nonretainedObjectValue;
Discussion

If the receiver can be interpreted as containing an object pointer, return that value. Otherwise, return nil.

pointerValue

- (void*) pointerValue;
Discussion

If the receiver can be interpreted as containing a pointer, return that value. Otherwise, return NULL.

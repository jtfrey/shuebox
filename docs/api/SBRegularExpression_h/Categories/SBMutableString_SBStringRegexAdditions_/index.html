SBMutableString(SBStringRegexAdditions)

Extends Class: SBMutableString
Declared In: SBRegularExpression.h

Discussion

Additional SBString methods which facilitate regular-expression-based find-and-replace operations.


Methods

-replaceAllMatchesForRegex:withString:

replaceAllMatchesForRegex:withString:

- (BOOL) replaceAllMatchesForRegex:(SBRegularExpression*)regex 
        withString:(SBString*)aString;
Discussion

Locate each range of characters in the receiver which matches the given regular expression and replace them all with the contents of aString. If the regular expression contains grouped ranges then the groups can be references as $1, $2, ... within aString.

replaceFirstMatchForRegex:withString:

- (BOOL) replaceFirstMatchForRegex:(SBRegularExpression*)regex 
        withString:(SBString*)aString;
Discussion

Locate the first range of characters in the receiver which match the given regular expression and replace that range with the contents of aString. If the regular expression contains grouped ranges then the groups can be references as $1, $2, ... within aString.

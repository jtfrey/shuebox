SBXMLElement(SBExtendedXMLElement) SBXMLElement

Extends Class: SBXMLElement
Declared In: SBXMLElement.h

Discussion

Instances of SBXMLElement represent an XML element, its attributes, and all nodes that are children of it. Any localized namespace declarations are recorded.

booleanAttributeForName:

- (BOOL) booleanAttributeForName:(SBString*)attribName;
Discussion

If the receiver contains the named attribute, attempts to transmute the value of the attribute into a boolean value. The text "yes" and "true" will return YES, as will any non-zero numerical value. All other values will return NO.

numberAttributeForName:

- (SBNumber*) numberAttributeForName:(SBString*)attribName;
Discussion

If the receiver contains the named attribute, attempts to transmute the value of the attribute into an SBNumber.

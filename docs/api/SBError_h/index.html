SBErrorUnderlyingErrorKey

extern SBString const* SBErrorUnderlyingErrorKey;
Discussion

Recommended standard way to embed additional SBErrors in an SBError "wrapper". The value for this key should be an SBError or an SBArray of SBErrors.

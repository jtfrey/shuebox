SBAcquirePIDFile

BOOL SBAcquirePIDFile(
    SBString *pidFile);
Discussion

Attempt to establish a PID file at the given path. If the file is empty or does not exist the call will be successful and return YES; the file will contain the pid until a call to SBDropPIDFile(). If unsuccessful, NO is returned.

SBDropPIDFile

void SBDropPIDFile(
    SBString *pidFile);
Discussion

Attempts to drop the pid stored in the pidFile.

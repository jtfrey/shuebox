SBDate(SBExtendedDate)

Extends Class: SBDate
Declared In: SBDate.h

Discussion

Groups all methods that extend the basic SBDate object. Subclasses of SBDate that at least override the utcTimestamp method automatically inherit these methods.

compare:

- (SBComparisonResult) compare:(SBDate*)anotherDate;
Discussion

Returns SBOrderDescending if anotherDate occurs before the receiver; SBOrderAscending if anotherDate occurs after the receiver; and SBOrderSame if the two are equivalent.

earlierDate:

- (SBDate*) earlierDate:(SBDate*)anotherDate;
Discussion

Returns the receiver itself if its UTC timestamp <= the UTC timestamp of anotherDate; otherwise, returns anotherDate.

icuDate

- (UDate) icuDate;
Discussion

Returns the ICU date which corresponds to the receiver's UTC timestamp.

laterDate:

- (SBDate*) laterDate:(SBDate*)anotherDate;
Discussion

Returns the receiver itself if its UTC timestamp >= the UTC timestamp of anotherDate; otherwise, returns anotherDate.

timeIntervalSinceDate:

- (SBTimeInterval*) timeIntervalSinceDate:(SBDate*)anotherDate;
Discussion

Returns an SBTimeInterval containing the interval between anotherDate and the date represented by the receiver. The interval will be negative if anotherDate is in the receiver's future.

unixTimestamp

- (time_t) unixTimestamp;
Discussion

Returns the Unix timestamp which corresponds to the receiver's UTC timestamp.

date

+ (SBDate*) date;
Discussion

Returns an autoreleased instance of SBDate which wraps the timestamp that was current when the instance was created.

dateWhichIsAlwaysNow

+ (SBDate*) dateWhichIsAlwaysNow;
Discussion

Returns a shared SBDate instance that is dynamic: whatever message you send to the object, it will always behave as though it wraps the current time.

This is distinct from the objects returned by the "date" class method: instances returned by that method are a static snapshot of the time when the instance was created.

dateWithICUDate:

+ (SBDate*) dateWithICUDate:(UDate)icuDate;
Discussion

Returns an autoreleased instance of SBDate which represents the given ICU date -- the number of milliseconds since 1970-01-01 00:00:00 UTC.

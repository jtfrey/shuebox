SBHTTPMethodFromString

SBHTTPMethod SBHTTPMethodFromString(
    SBString *aString);
Discussion

Parse aString and return the corresponding integral HTTP method identifier (from SBHTTPMethod). If aString does not contain a known HTTP method then kSBHTTPMethodINVALID is returned.

SBHTTPMethodToString

SBString* SBHTTPMethodToString(
    SBHTTPMethod method);
Discussion

Given an integral HTTP method identifier (from SBHTTPMethod), return an SBString containing the textual form of that method.

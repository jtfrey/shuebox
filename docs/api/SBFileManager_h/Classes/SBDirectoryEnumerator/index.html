SBDirectoryEnumerator

Superclass: SBEnumerator
Declared In: SBFileManager.h

Discussion

A public sub-class of SBEnumerator that is used for walking the contents of a directory. Each invocation of "nextObject" steps the receiver to the next child entity of the directory; this class's accessor methods operate on the in-scope child entity.


Methods

-isDirectory
-ownerGId
-ownerUId
-posixPermissions

isDirectory

- (BOOL) isDirectory;
Discussion

Returns YES if the in-scope child entity is a directory.

ownerGId

- (gid_t) ownerGId;
Discussion

Returns the Unix group id number of the group-owner of the in-scope child entity.

ownerUId

- (uid_t) ownerUId;
Discussion

Returns the Unix user id number of the user-owner of the in-scope child entity.

posixPermissions

- (mode_t) posixPermissions;
Discussion

Returns the permissions mask for the in-scope child entity.

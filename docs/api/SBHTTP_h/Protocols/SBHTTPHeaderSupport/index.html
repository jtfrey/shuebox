SBHTTPHeaderSupport

Declared In: SBHTTP.h

Discussion

Protocol which encapsulates methods that can be adopted by classes that wish to add HTTP headers to a HTTP session.


Methods

-appendHTTPHeaders:

appendHTTPHeaders:

- (void) appendHTTPHeaders:(SBMutableString*)content;
Discussion

The receiver should append any HTTP headers to the content string. If the receiver appends a blank line (signaling the end of headers) the behavior of the HTTP response will be undefined.

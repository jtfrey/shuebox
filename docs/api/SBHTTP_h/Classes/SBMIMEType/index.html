SBMIMEType

Superclass: SBObject
Declared In: SBHTTP.h

Discussion

Instances of SBMIMEType represent a MIME type. A MIME type consists of a media type and sub-type presented as two strings separated by a slash, a'la "text/plain". The MIME type may be augmented by key-value pairs delimited by a semi-colon:

text/plain; charset=utf-8; compressed=gz

initWithString:

- (id) initWithString:(SBString*)mimeString;
Discussion

Initializes the receiver to contain the parsed components of mimeString. If mimeString is not a valid MIME type string then the receiver is released and nil is returned.

mediaSubType

- (SBString*) mediaSubType;
Discussion

Return the receiver's media sub-type.

mediaType

- (SBString*) mediaType;
Discussion

Return the receiver's media type.

mimeTypeWithString:

+ (SBMIMEType*) mimeTypeWithString:(SBString*)mimeString;
Discussion

Returns an autoreleased instance that contains the parsed components of mimeString. If mimeString is not a valid MIME type string then nil is returned.

parameterForName:

- (SBString*) parameterForName:(SBString*)parameterName;
Discussion

If the receiver contains a MIME type parameter with the given parameterName, return its value. Otherwise, nil is returned.

parameters

- (SBDictionary*) parameters;
Discussion

Return the SBDictionary of MIME type paramters associated with the receiver.

SBCharacterSet(SBHTTPAdditions)

Extends Class: SBCharacterSet
Declared In: SBHTTP.h

Discussion

Additions to the SBCharacterSet class to provide support for character sets associated with the HTTP protocol.


Methods

+httpTokenCharacterSet

httpTokenCharacterSet

+ (SBCharacterSet*) httpTokenCharacterSet;
Discussion

Returns a character set whose membership is all characters valid in HTTP tokens.

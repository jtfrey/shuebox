SBDictionary(SBStringPairFiles)

Extends Class: SBDictionary
Declared In: SBDictionary.h

Discussion

Methods for creating dictionaries from files containing string-oriented key-value pairs.


Methods

+dictionaryWithStringPairFile:

dictionaryWithStringPairFile:

+ (id) dictionaryWithStringPairFile:(SBString*)path;
Discussion

Returns an newly-created, autoreleased dictionary which contains the key-value pairs contained in the file specified by path. Both key and value objects will be SBString instances.

Returns nil if the file could not be read or if there are no key-value pairs in the file.

SBMutableDictionary(SBExtendedMutableDictionary)

Extends Class: SBMutableDictionary
Declared In: SBDictionary.h

Discussion

Groups methods which extend the basic functionality of an SBMutableDictionary. All methods' default implementation is accessible by any proper subclass of SBMutableDictionary.

addElementsFromDictionary:

- (void) addElementsFromDictionary:(SBDictionary*)otherDictionary;
Discussion

Add all of the key-value pairs defined in otherDictionary to the receiver. For any keys in otherDictionary which are already present in the receiver, the value from otherDictionary will overwrite the existing value in the receiver.

If the receiver is a fixed-capacity mutable dictionary and the capacity is reached, the remaining key-value pairs from otherDictionary are not added.

removeAllObjects

- (void) removeAllObjects;
Discussion

Remove all key-value pairs in the receiver; the result will be an empty dictionary.

removeObjectsForKeys:

- (void) removeObjectsForKeys:(SBArray*)keyArray;
Discussion

Remove any key-value pairs in the receiver which have a key that matches (under the isEqual: method) an object in keyArray.

removeObjectsWithObject:

- (void) removeObjectsWithObject:(id)object;
Discussion

Remove all key-value pairs in the receiver which have a value that matches (under the isEqual: method) object.

setDictionary:

- (void) setDictionary:(SBDictionary*)otherDictionary;
Discussion

Removes all key-value pairs from the receiver and adds all key-value pairs from otherDictionary. If the receiver is a fixed-capacity mutable dictionary and the capacity is reached, the remaining key-value pairs from otherDictionary are not added.

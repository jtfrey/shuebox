+ (id) arrayWithObjects:(id)initialObject,...;
Discussion

Returns an autoreleased instance that contains initialObject plus all objects following it in the argument list. The list of objects should be terminated with a sentinel value of "nil", e.g. 

    [SBArray arrayWithObjects:obj1,obj2,obj3,nil];

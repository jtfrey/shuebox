isAbsolutePath

- (BOOL) isAbsolutePath;
Discussion

Returns YES if the receiver represents an absolute UNIX file path.

isRelativePath

- (BOOL) isRelativePath;
Discussion

Returns YES if the receiver represents a relative UNIX file path.

lastPathComponent

- (SBString*) lastPathComponent;
Discussion

Returns a string which contains the last component of the UNIX file path contained in the receiver. E.g.

["/opt/etc/ipf.conf" lastPathComponent] => "ipf.conf"

pathExtension

- (SBString*) pathExtension;
Discussion

Returns a string which contains the filename extension of the UNIX file path contained in the receiver. E.g.

["/opt/etc/ipf.conf" pathExtension] => "conf"

stringByAppendingPathComponent:

- (SBString*) stringByAppendingPathComponent:(SBString*)aString;
Discussion

Returns a string which contains the reciever with a "/" and aString appended to it. E.g.

["/opt/etc" stringByAppendingPathComponent:"ipf.conf"] => "/opt/etc/ipf.conf"

stringByAppendingPathComponents:

- (SBString*) stringByAppendingPathComponents:(SBString*)aString,...;
Discussion

Returns a string which contains the reciever with one or more strings suffixed using a "/" prefix on each. E.g.

["/opt/etc" stringByAppendingPathComponent:"ipf","ipf.conf",nil] => "/opt/etc/ipf/ipf.conf"

The list of arguments must be terminated by nil.

stringByAppendingPathExtension:

- (SBString*) stringByAppendingPathExtension:(SBString*)aString;
Discussion

Returns a string which contains the receiver with a "." and aString appended to it. E.g.

["/opt/etc/ipf.conf" stringByAppendingPathExtension:"bak"] => "/opt/etc/ipf.conf.bak"

stringByDeletingLastPathComponent

- (SBString*) stringByDeletingLastPathComponent;
Discussion

Returns a string which contains all but the last component of the UNIX file path contained in the receiver. E.g.

["/opt/etc/ipf.conf" stringByDeletingLastPathComponent] => "/opt/etc"

stringByDeletingPathExtension

- (SBString*) stringByDeletingPathExtension;
Discussion

Returns a string which contains all but the filename extension of the UNIX file path contained in the receiver. E.g.

["/opt/etc/ipf.conf" stringByDeletingPathExtension] => "/opt/etc/ipf"

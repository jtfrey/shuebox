SBOrderedSetComparator

typedef SBComparisonResult ( *SBOrderedSetComparator)(
    id objectFromSet,
    id extObject);
Discussion

Type of a function that is used to order two objects. The function should return an SBComparisonResult indicative of the order of extObject w.r.t. objectFromSet. E.g. if extObject should occur after objectFromSet, the function should return SBOrderedAscending.

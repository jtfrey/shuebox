SBOrderedSet

Superclass: SBObject
Declared In: SBOrderedSet.h

Discussion

An SBOrderedSet is a mutable array of objects. Objects are kept sorted in that array by means of a sorting method or function. Objects that sort equivalent to an object already in the array replace that object (making the array single-occupancy w.r.t. object values).

addObject:

- (void) addObject:(id)object;
Discussion

Attempts to insert object into the receiver's array. If the receiver's selector/comparator function indicates that object already exists in the array, it is replaced with the incoming object.

addObjectsFromArray:

- (void) addObjectsFromArray:(SBArray*)anArray;
Discussion

Attempts to insert the objects contained in anArray into the receiver's array. Behaves similar to "n" invocations of the addObject: method.

addObjectsFromSet:

- (void) addObjectsFromSet:(SBOrderedSet*)otherSet;
Discussion

Attempts to insert the objects contained in otherSet into the receiver's array. Behaves similar to "n" invocations of the addObject: method.

containsObject:

- (BOOL) containsObject:(id)object;
Discussion

Similar to indexOfObject:, but returns YES if the object was present.

count

- (SBUInteger) count;
Discussion

Returns the number of objects contained in the receiver's array.

indexOfObject:

- (SBUInteger) indexOfObject:(id)object;
Discussion

If object is present in the receiver's array, returns its index. Equality is established using the selector or comparator function associated with the receiver.

SBNotFound is returned if object was not present.

initWithArray:andComparator:

- (id) initWithArray:(SBArray*)array andComparator:(SBOrderedSetComparator)comparator;
Discussion

The receiver is initialized to initially contain the objects in array. Object ordering is established by means of a comparator function (see SBOrderedSetComparator).

initWithArray:andSelector:

- (id) initWithArray:(SBArray*)array andSelector:(SEL)selector;
Discussion

The receiver is initialized to initially contain the objects in array. Object ordering is established by means of a selector corresponding to an instance method that is prototyped as:

- (SBComparisonResult) compareToObject:(id)otherObject

initWithComparator:

- (id) initWithComparator:(SBOrderedSetComparator)comparator;
Discussion

The receiver is initialized to an empty set. Object ordering is established by means of a comparator function (see SBOrderedSetComparator).

initWithSelector:

- (id) initWithSelector:(SEL)selector;
Discussion

The receiver is initialized to an empty set. Object ordering is established by means of a selector corresponding to an instance method that is prototyped as:

- (SBComparisonResult) compareToObject:(id)otherObject

objectAtIndex:

- (id) objectAtIndex:(SBUInteger)index;
Discussion

Returns the object at the given index in the receiver's array.

objectEnumerator

- (SBEnumerator*) objectEnumerator;
Discussion

Returns an SBEnumerator that iterates over the objects in the receiver's array from first to last. Do not attempt to modify the receiver while using the enumerator!

removeAllObjects

- (void) removeAllObjects;
Discussion

Remove all objects from the receiver's array.

removeObject:

- (void) removeObject:(id)object;
Discussion

If object is present in the receiver's array, remove it. Equality is established using the receiver's selector/comparator function.

removeObjectAtIndex:

- (void) removeObjectAtIndex:(SBUInteger)index;
Discussion

Remove the object present at index in the receiver's array.

removeObjectsInArray:

- (void) removeObjectsInArray:(SBArray*)anArray;
Discussion

All objects in anArray which are found in the receiver's array are removed from the receiver's array. Equality is established using the receiver's selector/comparator function.

removeObjectsInSet:

- (void) removeObjectsInSet:(SBOrderedSet*)otherSet;
Discussion

All objects in otherSet which are found in the receiver's array are removed from the receiver's array. Equality is established using the receiver's selector/comparator function.

reverseObjectEnumerator

- (SBEnumerator*) reverseObjectEnumerator;
Discussion

Returns an SBEnumerator that iterates over the objects in the receiver's array from last to first. Do not attempt to modify the receiver while using the enumerator!

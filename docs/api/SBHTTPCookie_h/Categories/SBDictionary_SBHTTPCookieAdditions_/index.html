SBDictionary(SBHTTPCookieAdditions)

Extends Class: SBDictionary
Declared In: SBHTTPCookie.h

Discussion

Additions to the SBDictionary class to support HTTP cookies.


Methods

dictionaryWithCookiesFromEnv

+ (SBDictionary*) dictionaryWithCookiesFromEnv;
Discussion

Extract all cookies from the HTTP_COOKIE environment variable and return an SBDictionary keyed by the cookie names.

dictionaryWithCookieString:

+ (SBDictionary*) dictionaryWithCookieString:(SBString*)cookieString;
Discussion

Extract all cookies from cookieString and return an SBDictionary keyed by the cookie names.

initWithCookiesFromEnv

- (id) initWithCookiesFromEnv;
Discussion

Initialize the receiver to contain (keyed by name) all cookies present in the HTTP_COOKIE environment variable.

initWithCookieString:

- (id) initWithCookieString:(SBString*)cookieString;
Discussion

Initialize the receiver to contain (keyed by name) all cookies present in the cookieString.

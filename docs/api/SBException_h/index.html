_SBExceptionState

typedef struct _SBExceptionState { 
    jmp_buf origin; /* place to which execution will return on longjmp() */
    struct _SBExceptionState *link; 
    SBException *exception; 
} SBExceptionState;
Discussion

Consider this a private data structure! Each exception block places one of these on the stack and chains it to the previously-established exception blocks.

TRY_BEGIN

    { \

    _SBPushExceptionState(&SBLocalExceptionState); \ 
    if( ! setjmp(SBLocalExceptionState.origin) ) {
Discussion

Denotes the start of an exception block. Never explicitly return() inside an exception block -- see the TRY_RETURN_VALUE and TRY_RETURN macros!

TRY_RETURN

#define TRY_RETURN \ 
    do { \ 
    _SBPopExceptionState(); \ 
    return; \ 
    } while (0)
Discussion

Use this macro inside an exception block to return from the void-typed function. The exception state must be cleaned-up before exiting.

TRY_RETURN_VALUE

#define TRY_RETURN_VALUE(SBEXCEPTION_VALUE, SBEXCEPTION_VALUETYPE) \ 
    do { \ 
    SBEXCEPTION_VALUETYPE rc = (SBEXCEPTION_VALUE); \ 
    _SBPopExceptionState(); \ 
    return (rc); \ 
    } while (0)
Discussion

Use this macro inside an exception block to return a value from the function. The exception state must be cleaned-up before exiting. Takes two arguments: the value to return and its type. E.g.

TRY_RETURN_VALUE(@"Failed!", SBString*);

md5Digest:

- (BOOL) md5Digest:(const char*)digestString;
Discussion

Fills-in the 16-byte buffer at digestString with the MD5 digest hash corresponding with the UTF-16 representation of the receiver.

md5DigestForUTF8:

- (BOOL) md5DigestForUTF8:(const char*)digestString;
Discussion

Fills-in the 16-byte buffer at digestString with the MD5 digest hash corresponding with the UTF-8 representation of the receiver.

md5DigestForUTF8MatchesString:

- (BOOL) md5DigestForUTF8MatchesString:(SBString*)digestAsString;
Discussion

Returns YES if the MD5 digest of the receiver string (in UTF8 encoding) is equal to the 32-character digestAsString.

md5DigestMatchesString:

- (BOOL) md5DigestMatchesString:(SBString*)digestAsString;
Discussion

Returns YES if the MD5 digest of the receiver string is equal to the 32-character digestAsString.

SBMD5Digest

Superclass: SBObject
Declared In: SBMD5Digest.h

Discussion

Instances of this class are used to generate an MD5 digest for a stream of bytes.

- (BOOL) appendBytesToDigest:(const void*)bytes length:(SBUInteger)length;
Discussion

Process the given byte stream (length octets at bytes) into the receiver's current digest hash.

appendDataToDigest:

- (BOOL) appendDataToDigest:(SBData*)aData;
Discussion

Process the binary contents of aData into the receiver's current digest hash.

appendStringToDigest:

- (BOOL) appendStringToDigest:(SBString*)aString;
Discussion

Process the bytes of aString into the receiver's current digest hash. The string is processed as UTF-16 character data.

digestString

- (const char*) digestString;
Discussion

Marks the receiver as having completed its current digest action and returns a pointer to the 16-character MD5 digest string which was produced.

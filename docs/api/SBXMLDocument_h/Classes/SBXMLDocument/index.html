initWithData:

- (id) initWithData:(SBData*)data;
Discussion

Initialize a newly-allocated instance by parsing the bytes contained in data. The data is assumed to be UTF-8 character data OR have an lead-in with a specific encoding specified.

initWithRootElement:

- (id) initWithRootElement:(SBXMLElement*)rootElement;
Discussion

Initialize a newly-allocated instance to contain the given SBXMLElement as the document root.

initWithStream:

- (id) initWithStream:(SBInputStream*)stream;
Discussion

Initialize a newly-allocated instance by reading from the given stream and parsing that data.

initWithXMLString:

- (id) initWithXMLString:(SBString*)xmlString;
Discussion

Initialize a newly-allocated instance by parsing the text contained in xmlString.

isNamedDocument:

- (BOOL) isNamedDocument:(SBString*)rootElementName;
Discussion

Returns YES if the receiver's root element's name is equal to rootElementName.

isNamedDocument:namespaceURI:

- (BOOL) isNamedDocument:(SBString*)rootElementName 
        namespaceURI:(SBString*)namespaceURI;
Discussion

Returns YES if the receiver's root element's name is equal to rootElementName. The name must be qualified by the given namespace URI.

rootElement

- (SBXMLElement*) rootElement;
Discussion

Returns the root element of the receiver's XML document.

setRootElement:

- (void) setRootElement:(SBXMLElement*)rootElement;
Discussion

Sets the root element of the receiver's XML document to rootElement. If rootElement is nil, then the document is made "emtpy".
